Prompt para el desarrollo de la web personal de DogSoulDev

1. Temática y estilo:
- Web "oscura" con temática hacker y divertida.
- Impactante visualmente, con animaciones y efectos que sorprendan al usuario.
- Tipografía monoespaciada, colores oscuros, estilo terminal.

2. Contenido:
- Landing page que impresione y enganche.
- Portfolio con proyectos, tecnologías, enlaces y descripciones.
- Sección personal: resumen de pasión por informática, ciberseguridad, programación, deporte, etc.
- Reseñas de usuarios (como LinkedIn): formulario seguro para que otros dejen opiniones sobre trabajar/estudiar contigo.
- Enlaces a redes sociales (LinkedIn, GitHub, etc.).

3. Tecnologías y arquitectura:
- HTML5, CSS3 (sin frameworks externos), JavaScript ES6+ (Vanilla, sin dependencias).
- Opcional: Web Components para modularidad.
- Arquitectura MVC adaptada a frontend o basada en componentes.

4. Buenas prácticas:
- Principios SOLID, DRY, KISS.
- Código limpio, modular y bien documentado.
- Seguridad: validación y escape de entradas, CSP, evitar funciones inseguras.
- Escalabilidad y mantenibilidad: estructura modular, uso de GitHub para control de versiones.

5. Objetivo:
- Crear una web moderna, segura, escalable y sorprendente, que refleje tu experiencia y personalidad como experto en ciberseguridad y hacker ético.
- El código debe ser lo más limpio y sencillo posible, pero con un resultado visual y funcional que destaque.

6. Actualización continua:
- La web debe estar preparada para ser actualizada y mejorada fácilmente en el futuro.
